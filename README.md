# TUTO-GITHUB
Petit répertoire pour conserver quelque part les bonnes pratiques et usage de github en tant que débutant

# Dès que je démarre VSC pour travailler 
-   git pull
# Dès que j'ai fini et que je veux envoyer
- git add . (ne pas oublier le point)
- git commit -m "votre message"
- git push

# TUTO-LARAVEL
Lors ce que je lance un programme LARAVEL depuis LARAGON, ne pas oublier NPM INSTALL et NPM RUN DEV (puis à la fin NPM RUN BUILD)
